Tutorial
1Download and install with default options. YoloMouse should now be running as indicated by icon on your taskbar.
2Launch a game that uses a mouse cursor (not an FPS with reticle).
3Get to the part of the game where you see the cursor you want to change.
4Hit CTRL SHIFT ALT 1 (up to 9) to change that cursor. Hit same hotkey again to cycle through more cursors
5Most games have multiple cursors that change according to their context. Repeat steps 3-4 for each cursor you wish to change.
6To reset a cursor to the game default hit CTRL SHIFT ALT 0
7To decrease or increase cursor size use CTRL SHIFT ALT - or +
Problems
64 Bit GamesMany new games are 64 Bit and require the 64 Bit version of YM.
Run as AdminIf a game is running as administrator YM must also run as administrator. Some games, like Wildstar, require YM to be run as admin.
Hotkey ConflictsSome games, like Diablo 3, have CTRL SHIFT or ALT bound to change/hide cursor preventing the YM hotkey from assigning the intended cursor. Either temporarily rebind those keys in that game or in YM.
DPI ScalingIf you have this on disable it for games that don't support it properly like Diablo 3 and Heroes of the Storm.
Hardware CursorIf the hardware cursor setting is available in your game, ensure it is ON. This is standard in all modern games.
New Cursor DisappearsSome games, like Path of Exile, require you to restart the game after using YM with them for the first time.
Fullscreen modeIncreasing cursor size over 64 is not support in fullscreen mode. If you wish to use larger sizes switch to fullscreen-windowed or windowed.
Hotkeys
Ctrl Shift Alt 1-9Change active game cursor
Ctrl Shift Alt 0Reset current cursor to the game's default
Ctrl Shift Alt - or +Decrease or increase cursor size

Add Your Own Cursors
1Collect your .CUR or .ANI files and rename them to numbers between 0 and 89 (Example: 12.ANI).
2Copy them to C:\Program Files\YoloMouse\Cursors.
3Restart yolomouse.
4Test using CTRL SHIFT ALT 1-9. 1 for files 0-9, 2 for files 10-19, etc.
